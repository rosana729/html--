🔵 <header> y <head>
La cabeza oculta y el encabezado visible del sitio

📖 ¿Para qué sirven?

<head> es invisible para el usuario. Contiene el título de la pestaña, links a CSS, fuentes y metadatos para Google.

<header> es la zona visible de arriba: logo, nombre del sitio y menú principal.

<!-- HEAD: información para el navegador (no se ve) --> <head> <meta charset="UTF-8"/> <title>Mi Sitio Web</title> <link rel="stylesheet" href="estilos.css"/> </head> <!-- HEADER: encabezado visible del sitio --> <header> <img src="logo.png" alt="Logo"/> <h1>Nombre del sitio</h1> </header>

🟢 <nav> — Menú de navegación
La lista de links que guía al usuario por el sitio

📖 ¿Para qué sirve?

<nav> agrupa los links de navegación principales. Suele vivir dentro del <header> o del <footer>.

Dentro del <nav> se usa una lista <ul> con ítems <li> y enlaces <a>.

<nav> <ul> <li><a href="#">Inicio</a></li> <li><a href="#">Acerca</a></li> <li><a href="#">Servicios</a></li> <li><a href="#">Contacto</a></li> </ul> </nav>

🟠 <main>, <article>, <section> y <figure>
El corazón del contenido de tu página

📖 ¿Para qué sirve cada uno?

<main> — Contenedor del contenido principal. Solo debe haber uno por página.

<article> — Contenido independiente: una noticia, un post, una tarjeta.

<section> — Agrupación temática dentro del main o del article.

<figure> + <figcaption> — Imagen con su descripción.

<main> <article> <h2>Título del artículo</h2> <h3>Subtítulo</h3> <p>Contenido del artículo...</p> <figure> <img src="imagen.jpg" alt="Descripción"/> <figcaption>Pie de imagen</figcaption> </figure> <button>Leer más</button> </article> </main>

🟡 <aside> — Barra lateral
Contenido secundario: buscador, links relacionados, widgets

📖 ¿Para qué sirve?

<aside> contiene información complementaria al contenido principal: un buscador, links relacionados, publicidades, o un índice.

Suele mostrarse como una columna lateral usando CSS Grid o Flexbox.

<aside> <section> <h4>Buscar</h4> <input type="text" placeholder="Buscar..."/> </section> <section> <h4>Links relacionados</h4> <ul> <li><a href="#">Artículo 1</a></li> </ul> </section> </aside>

🩷 <footer> — Pie de página
Copyright, redes sociales y navegación secundaria

📖 ¿Para qué sirve?

<footer> cierra la página. Contiene el copyright, links secundarios, redes sociales y datos de contacto.

<footer> <p>© 2024 Mi Sitio. Todos los derechos reservados.</p> <nav> <a href="#">Inicio</a> <a href="#">Contacto</a> </nav> <ul class="redes"> <li><a href="#">📘 Facebook</a></li> <li><a href="#">🐦 Twitter</a></li> </ul> </footer>
